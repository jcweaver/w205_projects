# w205 Projects

A repository containing the 3 individual projects completed during the Spring 2021 term of the UC Berkeley MIDS course w205: Fundamentals of Data Engineering

## Project 1: Query Project
Practice with SQL, GCP, BigQuery, command-line tools, and Jupyter notebooks


## Project 2: Tracking User Activity
Practice with Kafka, Spark, HDFS, Docker, SQL, and Jupyter notebooks

## Project 3: Understanding User Behavior
Practice using Kafka, Spark, HDFS, Presto, Apache Bench, and Jupyter notebooks
